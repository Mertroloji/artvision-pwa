Hakkımızda

ArtVision Nedir?

ArtVision, dünya çapında önemli sanat eserlerini dijital ortamda kullanıcılarla buluşturmayı amaçlayan modern bir web uygulamasıdır. Sanat tarihine ilgi duyan bireylerin, farklı dönemlere ait eserleri kolay ve hızlı bir şekilde keşfetmesini sağlar.

Uygulamanın Amacı

Bu uygulamanın temel amacı, açık veri kaynaklarını kullanarak sanat eserlerini erişilebilir hale getirmek ve Progressive Web App (PWA) teknolojileri ile kullanıcı deneyimini geliştirmektir. ArtVision, hem çevrimiçi hem de çevrimdışı çalışabilen bir yapı sunar.

Kullanılan Teknolojiler

  • API: The Metropolitan Museum of Art Collection API
    https://metmuseum.github.io/
  • CSS Çatısı: Bootstrap 5
    https://getbootstrap.com/

Offline Çalışma Mantığı

ArtVision, Service Worker teknolojisi kullanarak temel uygulama dosyalarını (HTML, CSS, JavaScript ve ikonlar) önbelleğe alır. Canlı API erişilemediği durumlarda uygulama, yerel olarak saklanan örnek veriler (sample.json) üzerinden çalışmaya devam eder. Bu sayede kullanıcılar, internet bağlantısı olmasa dahi uygulamayı görüntüleyebilir.
